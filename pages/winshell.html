Windows

Shell Access

First time

  1. Download PuTTY 32 bit or 64 bit (Dalhousie computers already have this downloaded)
  2. Open PuTTY
  3. Enter "<csid>@bluenose.cs.dal.ca" in the hostname box, where <csid> is replaced by your csid.
  4. Make sure SSH is selected in the Connection Type Radio Buttons
  5. Enter "Bluenose" in the "Saved Sessions" box
  6. Press "Save" to save the session for the next time
  7. Press "Open" to connect
  8. Click yes to the Security alert. If you are worried about security, check to see if the fingerprint matches the line below starting with ssh-ed.
  9. Enter your csid password. No
  10. You can now issue shell commands to bluenose

Subsequent times

  1. Open PuTTY
  2. Select "Bluenose" from the list of saved sessions, and double click
  3. Enter your csid password. No
  4. You can now issue shell commands to bluenose

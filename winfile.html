Windows

File Access

First Time

  1. Download and install WinSCP (Dalhousie computers already have this downloaded)
  2. Launch WinSCP
  3. Select "New Site"
  4. Select "SFTP" under "File protocol"
  5. Enter "bluenose.cs.dal.ca" under "Host name"
  6. Enter the username you use on moodle under "User name"
  7. Enter the password you use on moodle in the "Password" field
  8. Press "Save" to store the session for next time.
  9. In the save dialog, press "OK" to save the session
  10. Press "Login"
  11. Accept the SSH key and store it in the cache
  12. You can now drag and drop files from windows into bluenose, or from bluenose into the file system
  13. Any files in the "public_html" folder will be accessible at http://web.cs.dal.ca/~<csname> where <csname> is the username you logged in with

Subsequent times

  1. Launch WinSCP
  2. From the site list, choose whatever you names your session the first time
  3. Press "Login"
  4. You can now drag and drop files from windows into bluenose, or from bluenose into the file system
  5. Any files in the "public_html" folder will be accessible at http://web.cs.dal.ca/~<csname> where <csname> is the username you logged in with

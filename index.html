Welcome to the Bluenose tips page. Here you will find instructions on how to connect to bluenose for issuing commands (shell access) or manipulating files (file access) for your operating system. Choose your operating system and type of access from the menu to the left.

Windows

Shell Access

First time

  1. Download PuTTY (Dalhousie computers already have this downloaded)
  2. Open PuTTY
  3. Enter "<csname>@bluenose.cs.dal.ca" in the hostname box, where <csname> is the name you use for logging in to moodle.
  4. Make sure SSH is selected in the Connection Type Radio Buttons
  5. Enter "Bluenose" in the "Saved Sessions" box
  6. Press "Save" to save the session for the next time
  7. Press "Open" to connect
  8. Store the new SSH key in the cache
  9. Enter the password you use on moodle
  10. You can now issue shell commands to bluenose

Subsequent times

  1. Open PuTTY
  2. Select "Bluenose" from the list of saved sessions
  3. Press "Load" to open the session
  4. Press "Open" to connect
  5. Enter the password you use on moodle
  6. You can now issue shell commands to bluenose

Windows

File Access

First Time

  1. Download and install WinSCP (Dalhousie computers already have this downloaded)
  2. Launch WinSCP
  3. Select "New Site"
  4. Select "SFTP" under "File protocol"
  5. Enter "bluenose.cs.dal.ca" under "Host name"
  6. Enter the username you use on moodle under "User name"
  7. Enter the password you use on moodle in the "Password" field
  8. Press "Save" to store the session for next time.
  9. In the save dialog, press "OK" to save the session
  10. Press "Login"
  11. Accept the SSH key and store it in the cache
  12. You can now drag and drop files from windows into bluenose, or from bluenose into the file system
  13. Any files in the "public_html" folder will be accessible at http://web.cs.dal.ca/~<csname> where <csname> is the username you logged in with

Subsequent times

  1. Launch WinSCP
  2. From the site list, choose whatever you names your session the first time
  3. Press "Login"
  4. You can now drag and drop files from windows into bluenose, or from bluenose into the file system
  5. Any files in the "public_html" folder will be accessible at http://web.cs.dal.ca/~<csname> where <csname> is the username you logged in with

Linux (Ubuntu)

These instructions are for Ubuntu, but should be similar for other Linux distros.

Shell Access

  1. Open the terminal
  2. Type "ssh <csname>@bluenose.cs.dal.ca" and press enter, where <csname> is the username you use on Moodle
  3. Accept the key fingerprint
  4. Type your password used on mooodle and press enter (no characters will appear while you are typing)
  5. You can now issue shell commands to bluenose

Linux (Ubuntu)

These instructions are for Ubuntu, but should be similar for other Linux distros.

File Access

  1. Go to the desktop, or any file window
  2. From the "Go" menu, select "Enter Location" (or press ctrl-L)
  3. Type "sftp://<csname>@bluenose.cs.dal.ca/users/cs/<csname>" and press enter, where <csname> is the username you use on Moodle
  4. (Optional) From the bookmark menu, select "Bookmark this location" or press ctrl-B
  5. You may now copy files in or out of this window like any other file window
